 TIM2_PWM — PA0, плавне дихання PA0 (-) і анаод до (+) резистр 330Om
 (максимум 25мА на пін STM32).

**Два регістри керують всім:**
```
ARR  = 999  → період (частота 1кГц)
CCR1 = 0-999 → duty cycle (0-100%)
```

**`CCMR1` — найзаплутаніший регістр:**
```c
TIM2_CCMR1 = (6 << 4) | (1 << 3);
// OC1M=110 → PWM mode 1
// OC1PE=1  → preload enable
```

**PA0 = TIM2 CH1** — не будь-який пін, тільки той що прив'язаний до каналу таймера. Це в datasheet, розділ Alternate Functions.

Зберігаємо `TIM2_PWM` в репо і що далі — **SPI + Nokia 5110** в окремому чаті?
