Підсумок для Місяця 3:
• 3 повних UART для периферії: U4→UART3, U3→UART1, U11→UART4
• SPI0 повний на гребінці U11 (піни 14-17) — для дисплеїв, flash, тощо
• SN74HC595 — керується через GPIO bit-bang (3 піни)
• SIM800L конектор — тільки TX, повний UART не виходить
• PIR конектор — НЕ підключати (пін 34 = RESET на Luckfox)
